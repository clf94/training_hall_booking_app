• This tool is to have an overview to see who comes at what time on which day in the hall.
  • If someone books a slot and wants to play, they must check-in to update the live occupancy so others can see that someone is playing. Don't forget to checkout once finished.
  • Bookings can only be deleted by an admin.
  • Every user can book for themselves or for a partner. If someone forgot to check-in/out, anyone else can do it instead.
  • To guarantee accessibility for everyone, only 1 hour per day per user is allowed (two 30-min slots or one full 1-hour slot). Book only if nobody else is using that slot.
  • Use check-in/check-out functionality responsibly. If your turn is over and the hall is full, please checkout to allow the next registered player to play.
  • You can book for the next two available days (Tue, Thu, Fri). For example, if today is Tue, you can book for today and Thu; if today is Wed, you can book for Thu and Fri.
